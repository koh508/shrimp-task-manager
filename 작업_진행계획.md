# 작업 계획: 온유 업데이트 테스트 및 시스템 상태 보고 스크립트 생성 (2026-03-17)
## 목표
SYSTEM/auto_scripts/ 폴더에 '온유 업데이트 테스트' 및 시스템 상태 보고 스크립트(test.py)를 생성하고 실행하여, CPU 사용량 90% 초과 시 경고 기능을 포함한 시스템 상태를 보고하며, CPU 부하를 최소화하는 방안을 고려한다.
## 단계별 계획
1. [ ] 단계1 — `SYSTEM/auto_scripts/test.py` 파일 생성: CPU 사용량, 메모리 사용량, 디스크 사용량을 측정하고 90% 초과 시 경고 메시지를 출력하는 Python 스크립트 작성. CPU 부하를 최소화하기 위해 `psutil` 라이브러리 사용 및 측정 간격 조절.
2. [ ] 단계2 — `test.py` 스크립트 실행: `execute_script` 도구를 사용하여 생성된 스크립트 실행.
3. [ ] 단계3 — 실행 결과 확인 및 보고: 스크립트 실행 결과를 확인하고 사용자에게 보고.
## 롤백 방법
`delete_file(filepath='SYSTEM/auto_scripts/test.py')`를 호출하여 생성된 스크립트 파일을 삭제한다.
## 예상 위험
- 스크립트 실행 중 오류 발생 가능성.
- `psutil` 라이브러리가 설치되어 있지 않을 경우 스크립트 실행 실패 가능성. (현재 환경에서는 기본적으로 설치되어 있다고 가정)
- CPU 부하 최소화 로직이 예상대로 작동하지 않을 가능성.
