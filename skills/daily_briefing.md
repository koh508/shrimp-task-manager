# 온유 스킬: 일일 브리핑

## 0단계: 데이터 수집 (동시 실행)
아래 4가지를 순서대로 빠르게 수집하라. 없으면 "없음"으로 표시.
1. `calendar_list()` — 오늘 일정
2. `task_list()` — 미완료 태스크 목록
3. `list_files("DAILY")` — 오늘 날짜 일기 파일 존재 여부 확인
4. `list_files("보완사항")` — 미처리 보완 사항 파일 존재 여부

## 1단계: 브리핑 포맷 출력
반드시 아래 구조를 엄수. 각 섹션이 비어있으면 "없음" 한 줄로 처리.

```
📅 오늘 브리핑 — YYYY년 MM월 DD일 (요일)

⚡ 오늘 핵심 미션 (최대 3개)
1.
2.
3.

📆 일정
- (calendar_list 결과)

✅ 미완료 태스크
- (task_list 결과)

📝 일기
- 오늘 일기: 작성됨 / 미작성

⚠️ 보완 사항 대기
- (미처리 보완사항 파일 수)

💬 한 마디
(용준 님 현재 목표 중 오늘 가장 중요한 것 한 줄)
```

## 2단계: 자체 검수
- [ ] calendar_list 실제 호출했는가?
- [ ] task_list 실제 호출했는가?
- [ ] 섹션 누락 없이 전부 출력했는가?
- [ ] 출력이 너무 길지 않은가? (10줄 이내 권장)

## 🚨 피해야 할 실수
- 도구 호출 없이 "일정 없음"으로 임의 출력
- 브리핑이 20줄 넘게 장황하게 작성
- "핵심 미션"을 5개 이상 나열 (최대 3개)
