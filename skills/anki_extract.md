# 온유 스킬: 안키 카드 추출

## 대상 영역 제한
- 공조냉동 (HVAC) 관련 폴더 파일
- OCU 소방방재 관련 폴더 파일
- 일기/감정 파일에서는 실행 금지

## 추출 기준 (이 3가지 패턴만 추출)
1. `=` 기호가 있는 공식 (예: Q = m × Cp × ΔT)
2. 숫자가 포함된 수치 데이터 (예: 표준 냉동 사이클 압축비 3~4)
3. "A는 B이다" 형태의 정의

## 출력 형식 (엄수)
- 반드시 `질문;정답` 형태로 출력 (세미콜론 구분)
- 질문은 30자 이내로 짧게
- 마크다운 코드블록(```) 절대 사용 금지
- 앞뒤 설명 문구 금지 — 카드 데이터만 출력

예시:
```
현열비 공식은?;현열/(현열+잠열)
표준 냉동 사이클 단계는?;압축→응축→팽창→증발
소화설비 작동 압력은?;0.1MPa 이상
```

## 저장 방법
추출 완료 후 SYSTEM/Anki_Ready.csv 파일에 append 모드로 저장.
저장 후: "📇 안키 카드 N개 추출 완료 → Anki_Ready.csv" 보고.

## 자체 검수
- [ ] 세미콜론(;)이 있는 줄만 저장했는가?
- [ ] 질문이 30자 이내인가?
- [ ] 일기/감정 파일에서 실행하지 않았는가?
