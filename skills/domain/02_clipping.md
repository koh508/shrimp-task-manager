# [DOMAIN SKILL] Clipping Rules

## 주간 추적 클리핑 규칙
사용자가 '일주일마다 추적', '매주 모니터링', '주간 추적', '주 1회 클리핑' 등을 요청하면:
1. 해당 주제를 5~10개의 세부 검색어로 분해한다
2. `set_weekly_clip(topics=[세부 주제 목록], action='add')` 호출
3. '✅ 주간 추적 등록 완료. 매주 1회 클리핑됩니다.' 안내
- 제거 요청 시: `set_weekly_clip(topics=[제거할 주제], action='remove')`

## 즉시 클리핑 요청 규칙
사용자가 '~로 클리핑 N개', '~에 대해 클리핑해줘', '~관련 클리핑' 등을 요청하면:
1. 해당 주제를 N개의 구체적인 세부 주제로 직접 생성한다 (N 미지정 시 기본 10개)
2. `set_clip_config(topics=[생성한 세부 주제 목록])` 저장
3. '✅ 주제 N개 설정 완료. 클리핑을 시작하려면 "클리핑 시작"이라고 말해주세요.' 안내

## 클리핑 주제 자연어 수정 규칙
사용자가 클리핑 주제를 추가/삭제/교체하려는 자연어 요청 시:
1. `get_clip_topics` 도구로 현재 주제 목록 가져옴
2. 요청에 따라 목록 수정
3. `set_clip_config(topics=[수정된 전체 목록])` 저장
4. '✅ 클리핑 주제 업데이트 완료. 현재 N개' 형식으로 알림
