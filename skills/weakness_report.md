# 온유 스킬: 약점 보강 리포트

## 0단계: 약점 탐지
아래 순서로 약점을 찾아라:
1. search_vault("어렵다 틀렸 이해 안") 호출
2. 오늘 날짜 일기에 "어렵다", "틀렸", "이해 안", "막혔" 키워드 탐색
3. 없으면: "오늘 기록된 약점이 없습니다. 직접 말씀해 주세요." 안내

## 1단계: 약점 개념 추출
- 발견된 문장에서 핵심 개념 키워드 1개 추출
- 예: "랭킨 사이클이 헷갈려" → 키워드: "랭킨 사이클"

## 2단계: 자문자답 검색
- analyze_trend 도구로 "{키워드} 비전공자도 이해하기 쉬운 비유와 암기법" 검색
- 검색 결과를 용준 님 맞춤형(짧은 순서도 + 키워드)으로 재가공

## 3단계: 리포트 저장
- 저장 경로: `99_Onew_Reports/약점보강_{키워드}_{날짜}.md`
- YAML 프론트매터 포함:
  ```yaml
  ---
  tags: [약점보강, {키워드}]
  날짜: YYYY-MM-DD
  author: Onew
  출처: 웹검색
  ---
  ```
- 본문 각 섹션 끝에 출처 인라인 표기 필수:
  - search_vault() 결과 → `[출처: Vault RAG]`
  - analyze_trend() 결과 → `[출처: 웹검색]`
  - Gemini 자체 지식 → `[출처: AI 생성] ⚠️ AI 생성 내용 — 사실 확인 권장`

## 4단계: 보고
"📚 약점 보강 리포트 저장 완료 → {경로}" 한 줄 보고

## 주의사항
- 하루 1회 실행 제한 (API 비용 절감)
- 같은 키워드 리포트가 이미 있으면 덮어쓰지 말고 날짜 추가
