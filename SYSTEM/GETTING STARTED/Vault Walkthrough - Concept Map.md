If you plan to work on very large projects this is what you should do ^soPm01eV

1. Open the Project Dashboard 
    - You can also do that with the hotkey:
      `Ctrl+Shift+P`

2. Press the settings beside the project name

3. Duplicate the Base Project

4. Select the Duplicate

5. Press the settings again and select 
   Edit Project

6. Update the project name to the project folder
   you want to work on

7. Update the Path by left clicking the textbox
   and select the appropriate folder ^ApT87aMW
