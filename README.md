## 프로젝트 설명

- 자동 오류 복구
- 학습 이벤트 기록
- 성능 최적화 (CPU/메모리 모니터링)
- Git 통합
- 보안: 권한 오류 복구

## 설치 가이드 (Installation)
1. Python 3.7 이상 설치.
2. 가상 환경: `python -m venv venv` (활성화: `source venv/bin/activate`).
3. 종속성: `pip install -r requirements.txt`.
4. 실행: `python mainsystem.py`.

## 프로젝트 구조 (Project Tree)
