## 기능 검증 항목

### 핵심 기능
- [ ] **퀴즈 모드**: `quiz_me` MCP 도구가 실제 Vault 파일에서 문제 출제
  - 테스트: "공조냉동 퀴즈 내줘"
- [ ] **파일 편집**: `edit_file` MCP 도구 정상 작동, 보호 파일 거부 확인
  - 테스트: 테스트 .md 파일 수정 요청
- [ ] **캘린더**: `calendar_add` → 구글 캘린더 반영 확인
  - 테스트: "내일 오전 10시 테스트 일정 추가해줘"
- [ ] **Vault 검색**: `search_vault` LanceDB 결과 정상 반환
  - 테스트: "냉동효과 관련 내용 검색해줘"

### 세션 관리
- [ ] **10턴 후 세션 리셋**: 10회 문답 후 자동 저장 + 맥락 유지 확인
- [ ] **대화기록 저장**: `대화기록/YYYY-MM/*_v2.md` 파일 생성 확인
- [ ] **세션 복원**: 재시작 후 이전 요약 컨텍스트 주입 확인

### ADHD 엔진
- [ ] **스트릭 기록**: "공부 완료" 입력 시 스트릭 카운트 증가 확인
- [ ] **도파민 메뉴**: "도파민 메뉴" 입력 시 응답 확인

### 안전장치
- [ ] **Safety 차단**: 민감 주제 입력 시 회사 모드에서 차단 확인
- [ ] **이중 실행 방지**: 두 번째 터미널 실행 시 경고 후 종료 확인
  - 테스트: v2 실행 중 다른 터미널에서 `python obsidian_agent.py` 재실행
- [ ] **smolagents tool loop 안정성**: max_steps(10) 초과 시 처리 확인

### 데이터 무결성
- [ ] **엔티티 merge**: v2 사용 후 `--legacy` 전환 시 entities.json 엔티티 보존 확인
- [ ] **시크릿 모드 저장**: v2에서 `시크릿 on` 후 재시작 시 모드 유지 확인
- [ ] **MCP subprocess 종료**: 온유 종료 후 좀비 프로세스 없음 확인
  - 확인: 종료 후 `tasklist | grep python`

## 롤백 기준

아래 중 하나라도 발생하면 즉시 `--legacy`로 전환:

- [ ] smolagents tool loop가 같은 도구를 무한 반복
- [ ] MCP 서버 연결이 10분 안에 끊어짐 (재연결 불가)
- [ ] API 사용량이 legacy 대비 2배 이상 (50턴 기준)
- [ ] 퀴즈 파일 출제 실패 3회 연속
- [ ] 캘린더 연동 완전 실패

롤백 명령:
```bash
python obsidian_agent.py --legacy
```
