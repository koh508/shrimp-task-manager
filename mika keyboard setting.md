# 블루투스 키보드 OS 모드 전환 요약

## 🔧 OS 모드 단축키

| 키보드 브랜드/모델 | Windows 모드 | Android 모드 | iOS/macOS 모드 | 초기화 |
|------------------|--------------|--------------|----------------|--------|
| 라이프썸         | Fn + Q       | Fn + W       | Fn + E         | Fn + ESC (4초) |
| 독거미           | Fn + W       | Fn + Q       | Fn + E / Fn + R | Fn + ESC (4초) |
| 일반형 (다수)     | Fn + Q       | Fn + W       | Fn + E         | Fn + R 또는 ESC |

> ⚠️ 모델마다 다르므로, 키보드 뒷면이나 설명서에 있는 단축키 확인이 가장 정확함

---

## 🛠️ Android에서 물리 키보드 레이아웃 설정

- **경로**:  
  `설정 → 일반 관리 → 언어 및 입력 → 물리 키보드`

- **설정 항목**:  
  - 연결된 키보드 선택  
  - 레이아웃 변경: `한국어 - 2벌식`, `영어 - QWERTY` 등

---

## 💡 한영 전환 팁 (Windows 기준)

- 기본 한영키: `오른쪽 Alt (R-Alt)` 또는 `Ctrl + Space`  
- 설정 경로:  
  `설정 → 시간 및 언어 → 언어 및 지역 → 한국어 → 옵션 → 키보드 레이아웃 변경`
