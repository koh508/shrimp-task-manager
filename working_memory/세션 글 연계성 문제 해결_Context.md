## 세션 글 연계성 문제 Context

현재 온유 에이전트가 생성하는 응답(글)들이 이전 대화 내용이나 사용자 세션의 맥락을 충분히 반영하지 못하고 있습니다.

**문제점:**
1.  **정보 단절:** 사용자가 여러 질문을 이어갈 때, 이전 질문의 답변이나 논의되었던 핵심 정보가 현재 응답에 반영되지 않아 대화가 뚝뚝 끊기는 느낌을 줍니다.
2.  **반복적인 정보 요청:** 온유가 이미 제공했거나 논의했던 내용을 다시 질문하거나, 사용자가 반복해서 이전 정보를 상기시켜야 하는 경우가 발생합니다.
3.  **비효율적인 작업 흐름:** 복잡한 멀티스텝 태스크를 수행할 때, 각 단계에서 이전 단계의 결과물이나 맥락을 자동으로 참조하지 못해 수동 개입이 필요해집니다.
4.  **사용자 경험 저하:** 일관성 없는 응답은 사용자가 온유를 "기억력이 없는" 에이전트로 인식하게 하여 신뢰도를 떨어뜨리고 효율적인 상호작용을 방해합니다.

**원인 (추정):**
*   **세션 관리 로직 부족:** 이전 대화 내용을 효과적으로 저장하고 참조하는 세션 관리 메커니즘이 미흡할 수 있습니다.
*   **프롬프트 엔지니어링 한계:** 현재 프롬프트가 이전 세션의 맥락을 주입하는 데 충분히 정교하지 않을 수 있습니다.
*   **컨텍스트 윈도우 한계:** 모델의 컨텍스트 윈도우 크기 문제나, 컨텍스트 내에서 관련 정보를 추출하는 능력의 한계일 수 있습니다.

**영향:**
*   작업 효율성 저하
*   사용자 불만 증가
*   온유 에이전트의 지능적 이미지 손상

**요청 사항:**
클로드코드에게 이 문제를 설명하고, 세션 연계성을 강화하기 위한 코드 수정 및 프롬프트 개선을 요청할 예정입니다.
