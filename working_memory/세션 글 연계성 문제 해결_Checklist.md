## 세션 글 연계성 문제 해결 체크리스트

### 클로드코드에 요청 전 확인 사항
- [ ] **현상 구체화:** 어떤 상황에서 세션 연계성이 부족하다고 판단했는지 구체적인 대화 예시나 시나리오를 준비했는가?
- [ ] **기대하는 결과 명확화:** 세션 연계성 개선을 통해 어떤 사용자 경험을 기대하는지 명확히 정의했는가?
- [ ] **잠재적 개선 방안 탐색:** (선택 사항) 자체적으로 생각해본 개선 방안이 있다면 정리했는가?
- [ ] **관련 파일/로그 준비:** 문제 현상을 설명하는 데 도움이 될 만한 관련 파일이나 시스템 로그가 있다면 준비했는가?

### 클로드코드 요청 후 확인 사항
- [ ] **수정 내용 이해:** 클로드코드가 제안하고 적용한 수정 내용을 완전히 이해했는가?
- [ ] **테스트 시나리오 준비:** 개선된 시스템의 세션 연계성을 검증할 수 있는 다양한 테스트 시나리오를 준비했는가?
- [ ] **테스트 실행 및 결과 기록:** 준비된 시나리오에 따라 테스트를 실행하고, 연계성 개선 여부 및 추가적인 문제점을 상세히 기록했는가?
- [ ] **피드백 및 추가 요청:** 테스트 결과를 바탕으로 클로드코드에게 구체적인 피드백을 제공하고, 필요한 경우 추가적인 개선을 요청했는가?

### 완료 기준

- [ ] 온유 재시작 후 미완료 작업 감지 확인
