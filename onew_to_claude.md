## 온유(Gemini RAG 에이전트) '공조냉동기계기사' 관련 답변 개선 요청

**문제 요약:**
온유가 '공조냉동기계기사' 관련 질문에 반복적으로 불충분하거나 부정확한 답변을 제공하고 있습니다. 특히 사용자가 특정 문서(예: 'OCU/2024 공조냉동기계기사 실기 합본') 내 특정 과년도 문제(예: '2023.8.22 시행 1번 문제')를 명시적으로 요청했음에도 불구하고, 관련 내용을 검색/제시하지 않고 AI 생성 문제를 보여주거나 일반적인 답변을 하는 문제가 심각합니다.

**문제 원인 추정:**

1.  **RAG Retrieval 실패:** '공조냉동기계기사' 관련 특정 문서를 정확히 색인하고 검색하는 기능이 미흡하거나, 특정 파일 경로/문제 번호와 같은 명시적 요청을 검색 쿼리로 제대로 변환하지 못하는 것으로 보입니다.
2.  **Intent Recognition 오류:** 사용자가 '특정 자료 검색'을 요청하는 의도를 '일반적인 정보 요청'으로 오인하여 생성(Generation) 모드로 전환되는 경향이 있습니다.
3.  **Retrieval 우선순위 낮음:** 명시적인 검색 요청이 있을 때, 검색(Retrieval)보다 생성(Generation)의 우선순위가 더 높게 설정되어 있을 가능성이 있습니다. 이는 RAG 시스템의 핵심 목적인 '접근 가능한 최신/정확한 정보 제공'을 저해합니다.
4.  **Context Window 활용 부족:** 긴 대화 흐름에서 사용자가 제공하는 이전 컨텍스트(예: "5천원 소모", "1번부터 14번까지 풀었다", "과년도문제 1번이 어려웠다")를 충분히 이해하고 다음 답변 생성에 활용하지 못하고 있습니다.

**구체적 개선 방향:**

1.  **명시적 검색 요청 처리 강화:** 사용자가 '과년도 문제', '실기 합본', '파일 경로/문제 번호 명시', '언급된 문서 내 검색' 등 특정 문서 또는 콘텐츠 검색을 요청하는 경우, 이를 최우선적으로 검색(Retrieval) 도구 사용으로 연결하도록 로직을 개선합니다.
2.  **'공조냉동기계기사' 자료 색인 및 접근성 강화:** 'OCU/2024 공조냉동기계기사 실기 합본'과 같은 중요 자료가 RAG 시스템에 정확히 색인되어 있고, 질의응답 시 해당 자료 내 특정 페이지 또는 문제 번호까지 효과적으로 검색될 수 있도록 합니다.
3.  **검색 실패 시 피드백 명확화:** 만약 요청한 특정 자료를 검색하지 못했을 경우, AI가 관련 자료가 없음을 명확히 안내하고 다른 접근 방식을 제안하도록 합니다 (예: "요청하신 2023.8.22 시행 1번 문제는 현재 자료에서 찾을 수 없습니다. 혹시 다른 연도나 유형의 문제를 찾으시나요?").
